Jira Core Support tools

This app is built on Python Flask framework.

written by Evandro Baginski - 2018

This app was built to put together the scripts support has made to help with some daily tasks, such as the import of users and groups via REST and also a script to pull worklogs at user level from a given Jira issue. It works based on the current Jira's API setup which, as all of us know, may change in the future, causing the app to break. Please, see below some of the requirements for the app to run properly, such as endpoints and CSV format.

All uploaded files are deleted from the server once they're no longer used by the scripts.

Worklogs

Worklogs are extracted from the full representation of the issue as JSON.
Endpoint: /rest/api/2/issue/<issue-key>

User import

Users will be imported via REST. The script reads the CSV file provided and makes API calls to Jira to create them accordingly.
Endpoint: /rest/api/2/user

The CSV file provided must respect the following format, including the header:

Username First name Last name Email address
foobar Foo Bar foo@bar.com

Group import

The scripts attempts to add the user into the given group. If the group doesn't exist, it creates it and then attempts to add the user into the group again.
Endpoint: /rest/api/latest/group/user?groupname=<username>

The CSV file provided must respect the following format, including the header:

Username Groupname
foobar Foo-bar-group
